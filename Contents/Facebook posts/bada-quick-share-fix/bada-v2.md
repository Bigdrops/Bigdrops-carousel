# Bada Quick Share Fix - Facebook Variation 2 (Nigerian Informal - Tech Breakdown)

See this thing. Quick Share does not work on Chinese ROM phones. No Google Play Services means no native file sharing. But there is a way now.

The app is called Bada. Open source, on GitHub. No account, no Play Store, no restrictions. Download the APK and install it. That is all.

Setup is straightforward. Download from GitHub releases, install, grant permissions. On Xiaomi, enable unrestricted background usage so the system does not kill it. Takes about 30 seconds.

It has been tested on the X17PM. Works with NearDrop on Mac and Quick Share on Windows. So Xiaomi Chinese versions, Oppo, Huawei can all share files with any device now.

Share this with someone who uses a Chinese ROM phone. They will thank you.
