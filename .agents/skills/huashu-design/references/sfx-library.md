## Usage Guidelines

### Volume Recommendations (from apple-gallery-showcase.md dual-track audio)
- **SFX Main Track**: `1.0` (no attenuation)
- **BGM Background Track**: `0.4 ~ 0.5` (SFX clearly cuts through)
- **Multiple SFX overlay**: Use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg Composition Template
```bash
# Single SFX aligned to time point:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection Decision Tree
1. **Tactile action** (typing/click/swipe) → `keyboard/` or `ui/`
2. **Element entry/exit** → `transition/`
3. **Container operations** (card/modal) → `container/`
4. **Status feedback** (success/failure/notification) → `feedback/`
5. **Progress/time passage** → `progress/`
6. **Brand landing/important moment** → `impact/`
7. **AI magic/transformation** → `magic/`
8. **Command line/code demo** → `terminal/`

### Avoid Sound Stacking
- Max **2 SFX** concurrent at the same time point
- Can use 3 when BGM is below 0.3
- Clear other SFX during brand impact (leave 0.2s gap before landing)

## Prompt Writing Principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Good prompt three elements**:
1. **Sound physics description**: what object, what action ("mechanical keyboard single key press")
2. **Texture/style qualifier**: apple-style / ios-style / cinematic / retro
3. **Negative exclusion**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
