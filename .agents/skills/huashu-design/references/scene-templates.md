## 1. Public Account Cover / Article Header Image

**Specifications**:
- Cover image: 2.35:1 (900×383px or 1200×510px)
- Body illustrations: 16:9 (1200×675px) or 4:3 (1200×900px)

**Key Design Elements**:
- Visual impact priority (users quickly scroll through feeds)
- Minimal or no text (public account title overlays on top)
- Moderate color saturation (WeChat reading environment tends toward white)
- Avoid excessive detail (must remain recognizable as thumbnail)

**Recommended Styles**: 01 Pentagram / 11 Build / 12 Sagmeister / 18 Kenya Hara / 07 Field.io

## 4. PPT / Keynote Presentation

**Specifications**:
- Standard: 16:9 (1920×1080px)
- Widescreen: 16:10 (1920×1200px)

**Key Design Elements**:
- One core message per page (no cramming)
- Clear type hierarchy (title 40pt+ / body 24pt+ / notes 16pt+)
- Generous whitespace for projection clarity
- Image-to-text ratio at least 60:40
- Consistent visual system (colors, fonts, spacing)

**Recommended Styles**: 01 Pentagram / 10 Müller-Brockmann / 11 Build / 18 Kenya Hara / 04 Fathom
